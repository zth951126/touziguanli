# 每日行业主题基金推荐系统

一款网页端的行业主题基金每日买卖推荐软件，覆盖 CPO/通信、黄金、AI、半导体、新能源、美股（纳指/标普/道指）、军工、红利央企等大方向，并在页面顶部展示重要国内外财经新闻。支持"我的持仓"功能，根据您的持仓成本和技术信号给出个性化买卖建议。

## 功能

- **行业主题基金推荐**：32 只核心 ETF，覆盖宽基、美股、AI、CPO/通信、半导体、机器人、游戏传媒、黄金、有色、煤炭、油气、银行、券商、地产、消费、医药、新能源、电力、军工、农业、红利、央企、跨境等方向
- **场外基金提示**：基于天天基金实时估值的短线操作提示
- **我的持仓**：添加自己的持仓（成本价、份额），系统结合技术评分与盈亏状态给出加仓/持有/减仓/止损建议
- **重要财经新闻**：顶部展示国内/国际财经要闻（新浪财经）
- **每日买卖策略**：基于均线、MACD、RSI、KDJ、布林带等指标生成评分与仓位建议
- **K 线详情**：点击查看近 30 日走势与完整指标

## 快速开始

```bash
# 进入项目目录
cd fund-stock-recommender

# 安装依赖
npm install

# 启动服务
npm start

# 浏览器访问
http://localhost:3000
```

首次访问页面时会自动拉取行情数据并计算推荐策略，大约需要 20-40 秒，请耐心等待。数据会缓存 5 分钟，5 分钟内再次刷新会更快。

## 推荐策略说明

系统基于以下技术指标综合打分：

| 指标 | 买入信号 | 卖出信号 |
|---|---|---|
| 均线排列 | MA5>MA10>MA20 多头排列 | MA5<MA10<MA20 空头排列 |
| 价格与 MA20 | 收盘价站上 MA20 | 收盘价跌破 MA20 |
| MACD | 金叉 / 多头区间 | 死叉 / 空头区间 |
| RSI(14) | 低于 30 超卖反弹 | 高于 70 超买 |
| KDJ | K 上穿 D 金叉 | K 下穿 D 死叉 |
| 布林带 | 触及下轨 | 触及上轨 |
| MA60 | 价格站上 MA60 | 价格跌破 MA60 |

根据综合评分给出操作建议：

- `+5 及以上`：强烈买入 / 重仓
- `+3 ~ +4`：买入 / 半仓以上
- `+1 ~ +2`：轻仓买入
- `0`：观望
- `-1 ~ -3`：减仓
- `-4 及以下`：卖出 / 清仓

## 我的持仓建议逻辑

结合技术评分与您的持仓盈亏状态：

- **技术看多 + 浮亏**：逢低加仓/补仓
- **技术看多 + 盈利**：继续持有
- **技术中性**：观望
- **技术走弱 + 盈利**：减仓止盈
- **技术走弱 + 浮亏**：关注止损/离场

持仓数据仅保存在浏览器本地（localStorage），不会上传到服务器。

## 数据来源

- A 股/ETF K 线：腾讯财经 / 东方财富公开接口（双源自动回退）
- 场外基金估值：天天基金公开接口
- 财经新闻：新浪财经滚动新闻接口
